매쉬ERP 개인정보 처리방침

시행일: 2026년 7월 1일  |  최종 수정: 2026년 7월 1일

핵심 원칙: 매쉬ERP는 모든 데이터를 사용자 기기와 사내 네트워크에만 저장합니다. 외부 서버로 데이터를 전송하거나 제3자와 공유하지 않습니다.

1. 수집하는 정보

매쉬ERP는 다음 정보를 브라우저 내부 저장소(IndexedDB)에만 저장합니다:

  • 거래처 정보 (상호명, 사업자번호, 연락처)
  • 상품 및 재고 정보
  • 견적서, 거래명세서, 거래 내역
  • 직원 정보 및 급여 데이터
  • 회계 분개 및 경비 내역
  • 일정 및 메모
  • DeskMesh 동기화 설정 (IP 주소, 로그인 토큰)

2. 정보 저장 방식

모든 데이터는 사용자의 로컬 기기에 저장됩니다:

  • 단독 사용: Chrome 확장의 IndexedDB (Dexie.js) — 기기 외부로 전송 없음
  • 팀 동기화: DeskMesh 서버 연결 시, 설정한 사내 네트워크 PC의 SQLite DB로 동기화. 인터넷 경유 없음

3. 제3자 정보 제공

매쉬ERP는 수집된 정보를 어떠한 제3자에게도 제공, 판매, 공유하지 않습니다. 외부 분석 서비스, 광고 네트워크, 클라우드 저장소를 사용하지 않습니다.

4. 네트워크 통신

확장 프로그램이 연결하는 네트워크 주소:

  • ws://*:39999 — DeskMesh 팀 동기화 (사용자가 직접 설정한 사내 IP, 선택 사항)

인터넷 외부로의 통신은 없습니다.

5. 권한 사용 목적

  • storage: IndexedDB 접근 (로컬 데이터 저장)
  • tabs, windows: 모듈별 전용 창 열기 (견적서, 재고관리 등)

6. 데이터 보관 및 삭제

데이터는 사용자가 Chrome 확장을 제거하거나 브라우저 데이터를 초기화할 때 함께 삭제됩니다. 매쉬ERP 자체적으로 데이터를 보관하거나 삭제하지 않습니다.

7. 아동 개인정보

매쉬ERP는 13세 미만 아동을 대상으로 하지 않으며 의도적으로 아동 정보를 수집하지 않습니다.

8. 개인정보 처리방침 변경

방침 변경 시 확장 버전 업데이트를 통해 공지합니다. 최신 방침은 이 페이지에서 확인할 수 있습니다.
