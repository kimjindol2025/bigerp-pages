자주 묻는 질문

데이터가 어디에 저장되나요?

모든 데이터는 Chrome 브라우저의 IndexedDB(내 컴퓨터)에만 저장됩니다. 서버로 전송되지 않습니다.

다른 컴퓨터에서 데이터를 쓰고 싶어요.

설정 → 백업/복원에서 JSON 파일로 내보낸 후 다른 컴퓨터에서 가져오기 하시면 됩니다.

확장을 삭제하면 데이터가 사라지나요?

네. 삭제 전 설정 → 백업에서 반드시 데이터를 저장해두세요.

원하는 기능이 없어요.
