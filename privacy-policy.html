개인정보처리방침

최종 업데이트: 2025년 · BigERP v1.9

📌 핵심 요약: BigERP는 사용자 데이터를 수집·저장·전송하지 않습니다.
모든 데이터는 사용자 브라우저(IndexedDB)에만 저장됩니다.

1. 수집하는 정보

BigERP는 어떠한 개인정보도 수집하지 않습니다.

  • 사용자 계정 없음
  • 서버 없음
  • 외부 데이터 전송 없음
  • 분석·추적 코드 없음

2. 데이터 저장 위치

사용자가 입력하는 모든 데이터(견적서, 거래처, 상품, 경비, 회계 분개 등)는 Chrome 브라우저의 IndexedDB에 저장됩니다. 이 데이터는 사용자 기기에만 존재하며, 어떤 서버나 외부 시스템으로도 전송되지 않습니다.

3. 사용하는 권한

  • storage — IndexedDB 데이터 저장 (로컬 전용)
  • tabs — 모듈별 새 탭 열기
  • sidePanel — 사이드패널 지원

브라우저 히스토리, 북마크, 쿠키, 비밀번호, 위치정보에 접근하지 않습니다.

4. 제3자 공유

수집 데이터가 없으므로 제3자와 공유하는 정보도 없습니다.

5. 데이터 삭제

Chrome 설정에서 사이트 데이터 삭제 또는 확장 프로그램 삭제 시 모든 데이터가 완전히 삭제됩니다. 설정 → 백업을 통해 미리 JSON 파일로 내보내기를 권장합니다.
